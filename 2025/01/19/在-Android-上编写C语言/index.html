在-Android-上编写C语言

本作主要推荐相关的app

c4droid 🎁🎁🎁

优点

  • 补全功能完善
  • 兼容性强（或许）
  • 编译器设置丰富

缺点

  • 输入时无符号辅助

CPP N-IDE 🎁🎁

优点

  • 纯粹

缺点

  • 英文
  • 符号单列
  • 没有补全等功能

Cxxdroid 🎁🎁🎁🎁

优点

  • 编译前纠错
  • 安全键盘

缺点

  • 符号辅助反人类
  • 卡卡的

C语言编译器 By 🎁🎁🎁🎁🎁

优点

  • 舒服
  • 国人开发，符合使用习惯

缺点

  • 很棒

C语言编译器IDE 🎁🎁🎁

  • 舒服，丝滑
  • 国人开发，符合使用习惯
  • 编译前纠错

缺点

  • 有严重的广告
